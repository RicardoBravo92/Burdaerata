# Burdaerata

Frontend application for Burdaerata - a Cards Against Humanity style party game built with Next.js, Tailwind CSS, and Clerk.

👉 **Backend Repository:** [https://github.com/RicardoBravo92/BurdaerataBackend](https://github.com/RicardoBravo92/BurdaerataBackend)

## Tech Stack

- **Next.js (App Router)** - React framework
- **Tailwind CSS** - Styling
- **Clerk** - Authentication
- **Sonner** - Toast notifications

## Getting Started

First, install dependencies and set up your `.env.local` file with your Clerk keys and the Backend API URL (`NEXT_PUBLIC_API_URL=http://localhost:8000`).

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
